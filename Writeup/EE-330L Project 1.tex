\section{Introduction}

This project was given as part of the Analog Electronics lab at SNHU. The goal of the project was to design
both an \ac{FSK} modulator and demodulator using a pair of \ac{PLL}s.

A \ac{PLL} is a type of circuit that at its core, measures and reduces the phase offset between two
incoming signals.

\autoref{fig:pll-block} shows a basic \ac{PLL} block diagram, in a way they are somewhat analogous to
op-amps. However, unlike an op-amp, the \ac{PLL} measures \emph{phase} offset between the input and 
output signals. This phase error is fed back to a \ac{VCO} to adjust its frequency until the phase 
error is minimized. This feedback is a \emph{negative} feedback loop, meaning that the 
produced error signal is inverse to the phase error.

\subsection{Input signal and Level Shifting}

I knew from the start that I wanted to design the FSK pair around transmitting some sort of simple, but
useful data. And with a suggested 5V rail signal, TTL-logic via USB was the most obvious choice.

There are many common baud rates for serial communication, I decided to design around 1200baud initially
but learned later that higher baudrates were possible by increasing the \ac{PLL}'s target frequency.
The \ac{PLL}'s loop bandwidth limits the maximum baudrate we can target, a good rule of thumb
for symbol rate is:

\begin{equation}
    f_\text{symbol} \leq \frac{1}{10} \cdot \text{Loop Bandwidth}
\end{equation}

and at 100kHz,

\begin{equation}
    f_\text{symbol} \leq \frac{1}{10} \cdot 100k = 10kHz \approx 9600baud
\end{equation}
